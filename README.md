# AI Spend Auditor

AI Spend Auditor is a SaaS cost analysis platform built for startup founders, engineering managers, and operations teams that want to understand where AI tool spending is increasing and how to optimize it. The platform audits subscriptions, compares pricing plans, estimates unnecessary spend, and generates actionable recommendations.

The tool is designed for early-stage startups and remote teams using multiple AI products like Cursor, ChatGPT, Claude, GitHub Copilot, Notion AI, and Perplexity. Users upload or manually enter team tool usage, and the system produces an audit report with estimated savings opportunities.

## Quick Start
### Clone the repository
### Install dependencies
npm install
### Run locally
npm run dev
